## 🙋 About

Hi, I'm **Saurabh Yadav**, an AWS Cloud Engineer at **K21 Technology Pvt. Ltd.** passionate about designing and deploying scalable, secure, and cost-effective cloud infrastructure on AWS.

This portfolio website was built from scratch using pure HTML, CSS, and JavaScript — no frameworks. It is containerized with Docker and deployed on **AWS** (S3 Static Hosting / EC2 + Nginx).

---

## ✨ Features

- ⚡ **Fully Responsive** — Mobile-first design that works across all screen sizes
- 🌙 **Dark / Light Mode Toggle** — Smooth theme switching with localStorage persistence
- ✍️ **Typing Animation** — Dynamic hero greeting with animated text cycling
- 👁️ **Visitor Counter** — Real-time visitor tracking integrated in the hero section
- 🎨 **Glassmorphism UI** — Modern card design aesthetic with blur effects
- 🎞️ **Micro-animations** — Smooth scroll reveals, hover effects, and transitions
- 📧 **Pre-filled Hire Me Email** — One-click recruiter outreach with a structured email template
- 📱 **Mobile Navigation** — Responsive hamburger menu for smaller screens
- 🐳 **Dockerized** — Ready to containerize and deploy anywhere
- 🔒 **SEO Optimized** — Meta tags, semantic HTML, and proper heading hierarchy

## ⚙️ Setup & Deployment

### Run Locally (Simple HTTP Server)

```bash
# Clone the repository
git clone https://github.com/Saurabh2600/saurabh-portfolio.git
cd saurabh-portfolio

# Serve with Python (or any static server)
python -m http.server 8080
```

Then open `http://localhost:8080` in your browser.

### Deploy to AWS

Refer to the detailed deployment guide: [AWS_DEPLOYMENT.md](./AWS_DEPLOYMENT.md)

> Hosted on **AWS S3** as a static website.
